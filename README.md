# Brazillian-Ecom-Analysis
end-to-end Data Analysis project using MySQL and Power BI to analyze 100k+ Brazilian e-commerce orders.
🌟 Project Overview
This project focuses on analyzing 100k+ orders from the Olist Store (Brazilian E-Commerce) from 2016 to 2018. The goal was to transform raw relational data into a high-fidelity, interactive dashboard to uncover revenue trends, seasonal patterns, and category performance.

🛠️ Tech Stack & Tools
SQL (MySQL): Performed data cleaning, handled null values, and validated complex metrics like Month-on-Month (MoM) growth using Window Functions.

Power BI: * Data Modeling: Established a Star Schema relationship between 7+ tables.

DAX: Created measures for Total Revenue and Order Counts.

UI/UX: Designed a minimalist, aesthetic layout with rounded containers and shadow effects for better readability.

📈 Key Business Insights
Total Revenue: Tracked a massive $13.59 Million in aggregate sales.

Seasonality: Identified May as the peak performing month ($1.50M revenue) and a significant dip in September ($0.62M), suggesting potential supply chain or seasonal marketing shifts.

Top Categories: Visualized high-grossing sectors using Top-N filtering to highlight major revenue drivers like Health & Beauty and Watches.

Regional Performance: Implemented dynamic City & State Slicers for granular drill-down into the Brazilian market.
